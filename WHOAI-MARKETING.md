# WhoAI — Google Analytics & Datadog for AI Agents

## AI agents are powerful — but they are burning your budget

Autonomous AI agents can now execute workflows, process data, and interact with systems 24/7.

But what happens when an AI agent gets stuck in a loop or uses an expensive model unnecessarily?

Most teams today have no way to:
- track real-time API spend per agent
- monitor token burn rates across different LLMs
- detect cost anomalies before the monthly bill arrives
- enforce daily or monthly budget limits

# Meet WhoAI

WhoAI is the cost observability and FinOps platform for autonomous AI systems.

It tracks every token, monitors every agent, and alerts you the moment spending spikes.

Instead of blindly paying massive API bills, teams gain:
- real-time cost visibility
- token tracking and analytics
- automated anomaly alerts
- a centralized agent registry

# One Simple Story

### Scenario:
A background Research Agent gets stuck in a recursive loop over the weekend, making thousands of calls to GPT-4o.

### What happens with WhoAI:

1. WhoAI monitors the agent's token usage perfectly in real-time.
2. Within 45 minutes, WhoAI detects a 400% spike in token burn rate.
3. The agent exceeds its daily budget threshold.
4. WhoAI triggers an immediate high-severity Slack alert.
5. The agent's API access is instantly halted via the Kill Switch.

**Result:**
You save thousands of dollars that would have otherwise been wasted on shadow AI spend.

# What WhoAI Provides

## Agent Registry
A unified dashboard showing every active agent, who owns it, and which environment it runs in.

## Real-Time Cost Analytics
Track spend by agent, spend by model, and overall spend velocity across your organization.

## Token Tracking
Monitor exact token usage to identify optimization opportunities (e.g., switching from GPT-4o to Claude 3.5 Haiku).

## Budget Monitoring & Alerts
Set daily and monthly limits. Get notified instantly when budgets are exceeded or anomalies occur.
