# WHOAI MVP LOCK

## Version
WHOAI v1.0

## Mission

WHOAI helps companies understand exactly where their AI spending is going.

Think:

"Google Analytics + Datadog for AI Agents"

NOT:

- AI Governance
- Compliance Platform
- AI Security Suite
- Approval Workflow System
- Policy Engine

Those ideas are intentionally out of scope for v1.

---

# Launch Goal

Launch within 30 days.

Get first paying customer.

Revenue matters more than feature completeness.

Every decision must help achieve one of these:

1. Get demo requests.
2. Get beta users.
3. Get paying customers.

If a feature does not directly support those goals, it is postponed.

---

# Target Customer

Primary:

- AI agencies
- AI startups
- Teams using OpenAI, Claude, Gemini
- Businesses spending $500+ per month on AI APIs

Secondary:

- Cursor power users
- Automation agencies
- Internal AI teams

---

# Core Problem

Companies receive AI bills.

Nobody knows:

- Which agent generated the spend.
- Which model caused the spend.
- Which team owns the spend.
- When spend suddenly spikes.
- Which agents are wasting money.

WHOAI solves this.

---

# Core Product

WHOAI tracks:

- AI Agents
- Token Usage
- Model Usage
- Spend
- Alerts

Nothing else.

---

# MVP Screens

1. Dashboard
2. Agents
3. Spend Analytics
4. Alerts
5. Settings

No additional screens before launch.

---

# Dashboard KPIs

- Total Spend
- Spend Today
- Token Usage
- Active Agents
- Active Alerts

Only these metrics.

---

# Database Models

Allowed:

- Organization
- User
- Agent
- SpendLog
- Alert

Everything else is optional after launch.

---

# Launch Criteria

Before launch WHOAI must:

- Show agent list
- Show spend history
- Show token usage
- Show model breakdown
- Show alerts
- Look professional

Nothing else is required.

---

# Success Metric

Success is NOT:

- More features
- More pages
- More code

Success is:

First paying customer.

---

# Rule

Whenever a new feature idea appears, ask:

"Will this help us get a paying customer in the next 30 days?"

If NO:

Add to backlog.

Do not build it.
